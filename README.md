# LLM-Reasoning-Epistemic-Alignment
Research Repository for Master's Thesis: Evaluating LLM reasoning and epistemic alignment
# Do Large Language Models Reason Like Humans? An Epistemic Alignment Perspective on Truthfulness

## Project Overview
This repository contains the conceptual framework and evaluation metrics for my Master's Thesis at Umm Al-Qura University. The research evaluates the reasoning mechanisms and cognitive architectures of state-of-the-art Large Language Models (LLMs) compared to human cognitive profiles, focusing on epistemic alignment, truthfulness, and hallucination reduction[cite: 10].
